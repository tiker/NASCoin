Installation Instructions

How to Install Nast Client

Download and Install Java		http://www.java.com

Ubuntu/Debian
1 - Download and Install Java		'sudo aptitude install openjdk-7-jre'
2 - Unzip 'nas_client.zip'		'unzip nas_client.zip'
3 - Go to your Nas Directory		'cd nas'
4 - Start Nas				run.sh or 'java -cp nxt.jar:lib/*:conf nxt.Nxt'

Windows
1 - Download and Install Java		Oracle JDK 7
2 - Unzip 'nas_client.zip'	
3 - Go to your Nxt Directory		'cd C:/PATH_TO_NAS_FOLDER'
4 - Start Nas				run.bat or 'java -cp nxt.jar;lib\*;conf nxt.Nxt'

Mac
1 - Download and Install Java		Oracle JDK 7
2 - Unzip 'nas_client.zip'	
3 - Go to your Nxt Directory		'cd nas'
4 - Start Nas				run.sh or 'java -cp nxt.jar:lib/*:conf nxt.Nxt'

Open Nas Client
- Open your browser and go to http://localhost:7875 or Wesley's Nas wallet http://localhost:7876
- Click the "Lock" icon in the upper left corner to unlock your account.
- Enter your passphrase or if you don't have an account, enter your new passphrase.  Make sure you use a very random, complex password of at least 80 characters.  You can use 0-1, a-z, A-Z, special and national characters.
